HTML Guide

This is a guide to learn/revise HTML

Headings

heading are of 6 sizes from 1 to 6,with h1 with largest font size. It needs a closing tag, tag for h1 is : <h1>"heading" </h1>

Paragraph

This tag is used to write paragraphs(needs closing tag) <p>

Anchor element

Example of anchorTag (directing to google.com)

This element is used to add hyperlinks to the page(<a href = " ">)(needs closing tag). The value assigned to its "href" is where this link directs you to.

Image tag

This tag is used to insert images in website.(does not need closing tag)(inline element)(<img src="" alt="">). The "scr" is used to define the source of image. It could be hyperlink,in computer directory etc. "alt" is used as text to be displayed in case image is not present.

Lists

there are 2 types of list(needs closing tag) - ordered(<ol>) and unordered(<ul>)
List items are represented by <li> tag

unordered list

  • item 1
  • item 2

ordered list

  1. item 1
  2. item 2

Nested List

  • item1
  • item2
    • item2.1
    • item2.2
  • item3
  • item4
    1. item4.1

Some Tags

div tag

This tag (<div>)is used to create division in html document for better readability and use to CSS.

br tag

This tag <br> is used to break paragraphs and jump into next line in a webpage.

hr tag

This tag <hr> is used to create a line break in html document
